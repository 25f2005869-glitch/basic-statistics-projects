## How to complete (recommended workflow)

1. **Download** the Excel file you want to work on.
2. **Open in Microsoft Excel** (best compatibility).
3. First, do **File → Save As** and create your own copy:
   - Example: `Activity 2.1 - YourName.xlsx`
4. Complete the activity:
   - Enter values only in the answer cells (don’t overwrite formulas).
   - Use Excel formulas when allowed (examples below).
5. **Check your work**:
   - Recalculate / refresh formulas (usually automatic).
   - Make sure all required questions are answered.
6. Submit/share the **completed copy** as required by your class.

## Useful Excel formulas (quick reference)

- Mean: `=AVERAGE(range)`
- Median: `=MEDIAN(range)`
- Mode: `=MODE.SNGL(range)`
- Standard deviation (sample): `=STDEV.S(range)`
- Variance (sample): `=VAR.S(range)`
- Min / Max: `=MIN(range)`, `=MAX(range)`
- Count: `=COUNT(range)`

## Troubleshooting

- **Formulas not working / showing as text**
  - Check if the cell format is “Text” and change it to “General”, then re-enter the formula.
- **Using Google Sheets**
  - Most formulas work, but formatting/charts may shift. Excel is preferred.
- **Accidentally changed formulas**
  - Re-download the original file and copy only your final answers into the fresh file.
